What is the point?

This is a self hosted web application which can be used to manage debt where cash is not practical or desirable. For example, as a form of community currency. It could also be used to keep track of chores within a family, student accommodation etc.

Is this like Bitcoin?

No. There is nothing de-centralized about this; you would need to trust the individual / group / organization running the virtual currency server.

Can I use it now?

Sure. But as beta software, there will likely be a few glitches to work out.
